# Topic: Oxygen 4.7 — Analiză Cross-Reference (2023)

## 1. Code Authoring: CodeMirror 6
Oxygen 4.7 a modernizat complet experiența de scriere a codului:
- **Autocomplete CSS & HTML:** Sugestii inteligente pentru proprietăți și tag-uri în style sheets și code blocks.
- **Mixed View:** O fereastră de tip "CodePen" care permite editarea simultană a PHP, HTML, CSS și JS într-un singur bloc de cod, accelerând dezvoltarea de micro-componente custom.
- **Code Folding:** Posibilitatea de a restrânge (collapse) secțiuni de cod pentru a menține spațiul de lucru curat.
- **Advanced Search/Replace:** Suport pentru Regex, Case-matching și căutarea în întregul fișier (inclusiv sub linia vizibilă).

## 2. Fix-uri Critice
- **Easy Posts Query:** Revenirea la "Default Query" ca setare implicită pentru noile presets, asigurând funcționarea corectă a arhivelor WordPress.
- **Navigation Target:** Corectarea suportului pentru atributul `target="_blank"` în Site Navigation, preluat corect din setările meniului WordPress.
