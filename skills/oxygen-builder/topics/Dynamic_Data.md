## Relații între fișiere:

### 1. Evoluție Tehnologică (Superseded)
- **Early Phase (2018)**: `Visually_Insert_Dynamic_Data...` și `Oxygen_In-Depth...` stabilesc fundamentele mapării datelor la atribute. Integrarea cu ACF se făcea manual prin `get_field()` în câmpul "PHP Function Return Value" (`A_Quick_Look_At_Oxygen...`).
- **Intermediate Phase (2019-2021)**: Introducerea suportului nativ pentru **ACF Repeaters** (`30_Repeater_Element`) și **MetaBox** (`Whats_New_In_Oxygen_39`). Fișierul `39_Easily_Display_Meta_Box...` este tutorialul tehnic direct pentru versiunea 3.9, eliminând necesitatea codului PHP manual pentru MetaBox.
- **Advanced Phase (2021-2026)**: Introducerea **Visual Query Builder** (`How_To_Visually_Build_WordPress_Queries...`) și a funcțiilor helper precum `oxy_fetch_url_param` (`Oxygens_Dynamic_Data_Show_Data...`). Tutorialele vechi despre query strings manuale sunt acum **superseded** de interfața vizuală.

### 2. Complementaritate (Workflow-uri Complete)
- **Design & Layout**: `Using_CSS_Grid_With_Repeaters_In_Oxygen` (2020) completează tutorialele de Repeater din 2019 prin adăugarea controlului modern de layout (CSS Grid vs Flexbox).
- **Interactive UI**: `Rebuilt_Creating_Video_Tabs...` (2021) folosește Dynamic Data (atribute custom `data-video`) combinat cu JavaScript pentru o experiență interactivă, extinzând utilizarea datelor dincolo de simpla afișare text/imagine.

### 3. Duplicare parțială / Redundanță
- Tutorialele de blog fundamental (`How_to_Build_a_WordPress_Blog...` și `How_To_Make_A_Featured_Blog_Posts...`) din 2019 conțin secțiuni aproape identice despre configurarea Repeater-ului și maparea titlului/imaginii, dar cu focus diferit (layout arhitectural vs sidebar widget).

## Concepte Cheie:

### A. Mecanismul "Insert Data"
- **Omniprezență**: Orice atribut (src imagine, text, background, URL buton) poate fi mapat.
- **Sursă PHP Custom**: Opțiunea "PHP Function Return Value" rămâne metoda de "last resort" pentru complexități neacoperite nativ (ex: calcule logice complexe).

### B. Repeater Element (Motorul de Loop)
- **Evoluție**: De la un loop vertical simplu la layout-uri complexe tip "grid-magazine" prin CSS și `nth-child` (`Using_CSS_Grid...`).
- **Suport ACF Pro/MetaBox**: Capacitatea de a face loop prin câmpuri de tip Repeater (nu doar posturi), centrală pentru site-uri cu date structurate (proprietăți, specificații).

### C. Advanced Query Builder (v3.8+)
- Permite query-uri complexe (meta_query, tax_query) direct în builder, eliminând necesitatea de a scrie array-uri PHP manuale.

### D. Helper Functions (2026)
- `oxy_fetch_url_param`: Esențială pentru filtrarea dinamică a listelor pe baza parametrilor din URL (ex: dintr-un form de search sau navigare pe categorii).

## Impact Oxygen 6.0+:
Oxygen devine un mediu de dezvoltare de tip "No-Code" avansat, unde logica de date care necesită de obicei cod custom (query-uri încrucișate, filtrare prin URL, loop-uri în loop-uri) este gestionată vizual, dar cu performanța WordPress (zero bloated code).
